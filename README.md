# ObservaStats

## Overview

ObservaStats is a lightweight, web-based platform for tracking and competing in group challenges. It's designed to bring teams together by turning daily activities into a friendly competition.

## Purpose

Create a fun way for groups to:
- Track personal and collective progress
- Compete fairly with transparent scoring
- Stay motivated through visible achievements
- View activity history and team statistics

## How It Works

Members join using access codes and log their daily activities. The system automatically calculates points, updates rankings, and displays results in real-time.

## Key Capabilities

**Simple Access**: One-time login with a unique code

**Flexible Logging**: Add activities with optional photos and descriptions

**Smart Scoring**: Automatic point assignment based on activity type and effort

**Live Leaderboards**: See your position and compare with teammates

**Activity Timeline**: Browse all team contributions and achievements

**Mobile Friendly**: Works seamlessly on phones, tablets, and desktops

## Perfect For

- Study groups
- Fitness challenges
- Reading clubs
- Team building events
- Community competitions

## No Setup Required

Just share the link and access codes with your group. Everyone can start logging immediately.
